# PPAPI
Implementation of webAPI using Flask
## POST

### example 1
body
```
{
    "arg1": "pen",
    "arg2": "apple"
}
```
response
```
{
    "Error": false,
    "PPAP": "I have an aapen! I have an apple! oh appleaapen!"
}
```
### example 2
body
```
{
    "arg1": "banana",
    "arg2": "hogehoge"
}
```

response
```
{
    "Error": false,
    "PPAP": "I have a banana! I have a hogehoge! oh hogehogebanana!"
}
```

This API is available on  https://ppapi-nkgwer.herokuapp.com/
